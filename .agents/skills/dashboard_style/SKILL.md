# Butter & Cherry Design System for HybridRouter Dashboard

Use this system for all styling, styling tokens, components, and layouts of the dashboard frontend.

## 🎨 Theme Colors
*   **Primary Accent (Cherry):** `#75070C` (Used for main branding, key interactive highlights, prominent buttons, headers, and active states).
*   **Secondary Background/Accent (Butter):** `#FFEDAB` (Used for cards backgrounds, subtle borders, warm highlights, and light-mode panels).
*   **Theme Vibe:** A rich, high-contrast, premium aesthetic blending warm Butter tones with deep, elegant Cherry red.

## ✍️ Typography
*   **Headers & Titles:** **Ancola** (A bold, distinct display font for <h1> to <h4> elements to give the dashboard a premium, custom-branded identity).
*   **Body & UI Text:** **Saira** (A highly readable, clean sans-serif font for table content, metrics, charts, and descriptions).

## 🚀 Component Guidelines
*   **Micro-animations:** Always use `framer-motion` for transitions, card hover scales, and data list reveals.
*   **Charts:** Use `recharts` for visualization, styling the area strokes and tooltips with gradients utilizing Cherry red and warm Butter shades.
*   **Layout:** Keep layouts responsive, utilizing flexbox/grid with soft card borders, warm dropshadows, and clear spacing.
